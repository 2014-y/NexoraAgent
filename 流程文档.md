   快捷方式目标: C:\Windows\System32\cmd.exe /k C:\Users\Yuan\.openclaw\start-gateway.bat

## 2. start-gateway.bat 执行

   文件: C:\Users\Yuan\.openclaw\start-gateway.bat

   步骤:

1. setlocal EnableDelayedExpansion

2. 设置 NVM 环境变量 (C:\Users\Yuan\AppData\Roaming\nvm)

3. call nvm use v24.13.0  -> 切换到 Node.js v24.13.0

4. 检测端口 18789 是否有旧 Gateway 在运行
   
   - 有 -> taskkill 杀掉旧进程
   - 没有 -> 继续

5. 设置 NODE_HOME=C:\Users\Yuan\AppData\Roaming\nvm\v24.13.0

6. cd /d C:\Users\Yuan\.openclaw

7. 执行: node gateway run
   
   ->

## 3. gateway.cmd 执行

   文件: C:\Users\Yuan\.openclaw\gateway.cmd

   设置环境变量:

- HOME=C:\Users\Yuan

- TMPDIR=C:\Users\Yuan\AppData\Local\Temp

- OPENCLAW_GATEWAY_PORT=18789

- OPENCLAW_SERVICE_MARKER=openclaw

- OPENCLAW_SERVICE_KIND=gateway

- OPENCLAW_SERVICE_VERSION=2026.6.10
  
  执行: node node_modules\openclaw\dist\index.js gateway --port 18789
  
  ->

## 4. OpenClaw Gateway 启动流程

   入口: node_modules\openclaw\dist\index.js

   4.1 加载配置文件
       读取: C:\Users\Yuan\.openclaw\openclaw.json
       验证: config validate

   4.2 加载模型配置
       - agnes-ai: agnes-2.0-flash (主), agnes-1.5-flash (备)
       - yitong (通义千问): qwen3-max
       - ollama (本地): jarvis, jarvis-learned, jarvis-enhanced, qwen2.5, my-custom-ai

   4.3 加载插件 (32个)
       自定义插件:
       - dual-model-trainer (双模型教学)
       - weixin-reconnect (微信自动重连)
       - workboard (任务看板)

       内置插件: browser, memory-core, ollama, discord, feishu,
       duckduckgo, searxng, sms, webhooks, irc, bonjour,
       openclaw-weixin, voice-call, llm-task 等

   4.4 加载 Workspace 系统提示词 (注入到 system prompt)
       - AGENTS.md (8308 chars) - 行为规范
       - SOUL.md (已清空) - 性格设定
       - MEMORY.md (687 chars) - 长期记忆
       - IDENTITY.md (已清空) - 身份信息
       - USER.md (已清空) - 用户信息
       - TOOLS.md (860 chars) - 工具笔记
       - HEARTBEAT.md (168 chars) - 定时任务

       总系统提示词约 41000 字符

   4.5 启动 HTTP Server
       - 监听: http://127.0.0.1:18789
       - WebSocket: ws://127.0.0.1:18789
       - Auth Token: feishu-bridge-local-d897fd501f04c3134f875de331f43ed1

   4.6 启动 Channels
       - openclaw-weixin (微信): ON, OK
         - notifyStart 通知微信服务器
         - getUpdates long-poll 监控循环启动
         - 恢复上次 sync buf (104 bytes)
       - whatsapp: ON, SETUP

   4.7 启动 Health Monitor (300s 间隔)

   4.8 Gateway Ready - 等待消息

   ->

## 5. 用户通过微信发送消息

   5.1 微信服务器 -> OpenClaw (getUpdates long-poll)
       POST https://ilinkai.weixin.qq.com/ilink/bot/getupdates
       timeoutMs: 35000

   5.2 OpenClaw 收到消息
       - 解析: from_user_id, context_token, item_list
       - 记录日志: inbound message

   5.3 消息路由
       - Channel: openclaw-weixin
       - Account: 385aea8dd4fa-im-bot
       - Agent: agent:main:main (直接对话)

   5.4 插件钩子 onMessage (dual-model-trainer)
       将消息存入 context.dualModelData

   5.5 模型推理
       - 主模型: agnes-2.0-flash (128k 上下文)
       - 构建 prompt: [系统提示词] + [对话历史] + [用户问题]
       - 调用 API:
         POST https://apihub.agnes-ai.com/v1/chat/completions

   5.6 模型返回回复

   5.7 插件钩子 onAfterResponse (dual-model-trainer)
       - 老师模型示范回答 -> 学生模型模仿口吻 -> 保存训练数据
       - 或老师不可用时学生自主回答

   5.8 消息发送
       POST https://ilinkai.weixin.qq.com/ilink/bot/sendmessage
       Body: { to: xxx, text: 回复内容 }

   5.9 微信推送给用户手机

   ->

## 6. weixin-reconnect 插件持续监控

   每 30 秒查询一次 channel 状态
   超过 3 分钟无活动 -> 自动重启 channel
   重连成功 -> 重置计数器
