# openclaw.json 配置模板

复制此文件为 openclaw.json 并填入你的 API Key。

## 获取 API Key
1. 访问 https://agnes-ai.com/zh-Hans/docs/agnes-video-v20
2. 注册/登录账号
3. 在控制台创建 API Key

## 配置步骤
1. 复制: copy openclaw.json.example openclaw.json
2. 编辑: 将 YOUR_AGNES_API_KEY 替换为你的实际 Key
3. 可选: 修改端口、模型、插件等配置

## 注意事项
- 不要将含真实 API Key 的配置文件提交到 Git
- .gitignore 已排除敏感文件
- 建议使用环境变量存储密钥
