## 1. 接入前必看

请先确认：

| 检查项 | 正确状态 |
| :--- | :--- |
| Nexora Agent 已打开 | 窗口在运行 |
| 服务已启动 | 状态灯 **绿色** |
| 模型可用 | 「模型会话」里发一句能收到回复 |
| 手机微信能上网 | 能正常扫一扫 |

绿灯 + 模型能聊，再绑微信，成功率最高。

## 2. 方式 A：在软件里扫码绑定微信（推荐）

这是普通用户的主路径，**一般不需要命令行**。

### 步骤

1. 打开 Nexora Agent，确认已是 **绿灯**。  
2. 点左侧 **「通讯管理」**（或带「微信 / 渠道」字样的页面）。  
3. 找到 **微信** 卡片。  
4. 点击 **「扫码绑定」**（或同义按钮）。  
5. 中间日志区 / 弹层会出现 **二维码**。  
6. 打开手机微信 → **扫一扫** → 扫描电脑上的二维码。  
7. 在手机上确认登录 / 授权。  
8. 回到电脑，看微信卡片是否变为 **「已绑定」**（或绿色已连接状态）。

### 绑定成功长什么样？

- 卡片状态从「未绑定 / 未配置」变成「已绑定」  
- 日志里可能出现登录成功、账号 id 一类提示  

如果一直转圈、没有二维码：见下方[常见问题](#6-常见问题)。

---

## 3. 绑定成功后怎么测试

1. **保持电脑端 Nexora Agent 开着且绿灯。**  
2. 用手机微信（绑定的那个号，或另一台设备上的微信）给助手发：  
   `你好`  
3. 等待几秒，看是否收到 AI 回复。

### 测不通时怎么判断问题出在哪？

| 现象 | 更可能的原因 |
| :--- | :--- |
| 「模型会话」也不回 | API Key / 网络问题 |
| 会话能回，微信不回 | 绑定未成功、通道掉线、软件被关 |
| 偶尔回、经常断 | 电脑休眠、软件退出、网络不稳 |

## 4. QQ / 飞书等其他渠道

操作与微信类似，都在 **「通讯管理」** 里：

1. 切换到对应卡片（QQ、飞书等）  
2. 按提示扫码或填写平台要求的 Bot Token / 应用凭证  
3. 绑定成功后，用该 App 发消息测试  

不同平台要求不同（有的要在开放平台建机器人），以软件内说明和平台官方文档为准。

**注意：**

- 一个渠道绑定成功后，换号通常需要重新扫码 / 重新授权  
- 不要同一时间用多个地方反复登录同一个账号，容易互相挤下线

## 5. 只让特定人能找你的助手（可选）

默认策略因版本和配置而异：有的环境会对陌生人配对做限制。

如果你在界面里看到「配对 / 白名单 / 批准联系人」相关入口：

1. 陌生人第一次找助手时，系统可能给出配对码  
2. 你在电脑端批准后，对方才能继续聊  

若你暂时只用自己和信任的人测试，保持默认即可。

> 进阶用户若熟悉 OpenClaw 命令行，也可使用 `openclaw pairing ...` 一类命令管理授权；普通用户优先用软件界面。

## 6. 常见问题

### Q1：点了扫码，一直没有二维码？

按顺序排查：

1. 状态是不是 **绿灯**？不是就先启动。  
2. 网络是否正常？公司网/校园网有时拦登录接口。  
3. 停止后再启动一次，再点扫码。  
4. 看「系统日志 / 控制台」有没有红色报错。  

### Q2：扫了码，手机显示成功，电脑仍是未绑定？

- 等 10～20 秒看卡片是否刷新  
- 停止服务再启动一次  
- 重新点一次扫码绑定（旧码可能过期）  

### Q3：昨天还能聊，今天微信没反应了？

最常见原因：

- 电脑休眠 / 关机了  
- Nexora Agent 没开，或服务停了（红灯）  
- 微信被挤下线，需要重新扫码  

处理：打开软件 → 启动到绿灯 → 必要时重新扫码绑定。

### Q4：会不会泄露我的微信账号？

- 扫码登录凭证保存在本机相关配置 / 状态目录中  
- 请勿把含登录信息的配置文件分享给别人  
- 也不要在网吧 / 公共电脑上扫自己的主微信号  

### Q5：能同时绑多个微信号吗？

视当前版本与插件能力而定。一般可以在通讯管理里管理多账户；以界面是否提供「添加账号 / 多账户」入口为准。

### Q6：我不想用微信，只要电脑里聊

完全可以。用左侧 **「模型会话」** 就够了，不必绑任何渠道。
