## 1. 接入前必须确认的准备工作

开始绑定前，请花费半分钟对照检查，确认您的软件和环境处于以下状态：

| 检查项目 | 正确的健康状态 |
| :--- | :--- |
| **软件运行状态** | Nexora Agent 已经打开，且没有闪退 |
| **本地服务指示灯** | 界面左上角的指示灯为 🟢 **绿色 (已就绪)** |
| **AI 模型可用性** | 在左侧「模型会话」里发一句 `你好` 能正常收到 AI 的回答 |
| **手机与网络** | 手机能够正常联网，且摄像头能正常进行扫码操作 |

> [!IMPORTANT]
> **只有当“本地绿灯亮起”且“模型会话能回话”时，再去扫码绑定微信，成功率才最高。** 否则即使绑定了，AI 助手也会因为没有大脑或者服务未开启而无法回复消息。

## 2. 微信扫码接入（最推荐、最常用）

这是普通用户最主要的使用方式，**整个流程纯界面操作，完全不需要使用任何命令行。**

### 👣 详细绑定步骤：
1. 双击打开桌面上的 **Nexora Agent**，确保服务处于 🟢 **绿色就绪** 状态。
2. 在左侧菜单栏中点击 **「通讯管理」**。
3. 找到名为 **微信** 的渠道卡片。
4. 点击卡片上的 **「扫码绑定」** 按钮。
5. 稍等 2 到 5 秒，软件正下方的日志控制台内或者界面弹窗中会刷新生成一个临时的**登录二维码**。
6. 打开您准备用作 AI 助手的手机微信，点击右上角的 **扫一扫**，扫描电脑屏幕上的二维码，并在手机上点击确认授权登录。
7. 扫码并确认后，回到电脑软件界面，您会发现微信卡片的状态已经更新为 **「已绑定」**，代表微信助手已经正式上线！

## 3. 如何进行聊天测试与日常维护

### 🧪 快速测试是否成功：
1. **请保持您的电脑处于开机状态，且 Nexora Agent 软件保持打开并绿灯运行。**
2. 用**另外一个**手机或微信账号，向您刚刚扫码绑定的这个助手微信号发送一条消息，比如：`你好，介绍一下你自己。`
3. 等待几秒钟，如果这个助手微信号自动给对方发送了 AI 生成的回复，说明绑定与测试完美成功！

### 🔍 怎么判断是哪里出了问题？
如果测试时发现发消息没有自动回复，请参考下表排查：

| 故障现象 | 可能的原因 | 对应的排查步骤 |
| :--- | :--- | :--- |
| **助手微信号收到消息了，但完全不回复** | 1. 软件的「模型会话」能聊天吗？如果不可以，说明是 **API Key** 填错或失效了。<br>2. 软件左上角的指示灯变红了？说明服务掉线了，需要重新点启动。 |
| **发送消息后直接提示发送失败** | 助手微信号可能在手机端或者其他地方被动退出了登录，通道已失效。请在「通讯管理」里重新点一次扫码绑定。 |
| **AI 回复断断续续，不稳定** | 您的电脑可能进入了**休眠或睡眠状态**。请在 Windows 的电源设置中，将电脑的睡眠时间设置为“从不”，保证后台网关能一直联网运转。 |

## 4. 接入飞书 / QQ 等其他渠道

如果您除了微信外，还想让 AI 在飞书或者 QQ 上帮您干活，操作方法大同小异：

1. 打开左侧的 **「通讯管理」** 面板。
2. 找到 **飞书** 或 **QQ** 的对应卡片。
3. **飞书接入**：您需要在飞书开放平台创建一个企业自建应用，然后按照软件界面的指引，填入 `App ID` 和 `App Secret` 凭证完成连接。
4. **QQ 接入**：按照卡片提示扫码登录或者配置官方的机器人密钥。
5. 绑定成功后，使用对应的聊天工具发送消息进行测试即可。

> [!TIP]
> **温馨提示：**
> 请不要用多个地方或者多台电脑同时扫码绑定同一个微信/QQ账号，这会导致账号登录状态冲突，容易被系统强制踢下线。

## 5. 进阶玩法：设置配对码与白名单（可选）

为了防止不认识的陌生人狂刷您的 AI 额度，或者在群聊中造成骚扰，您可以使用授权配对机制（白名单功能）：

- **陌生人配对**：当一个未授权的微信号第一次给您的 AI 发送消息时，AI 助手不会直接回答，而是会返回一个带有配对码的提示。
- **允许聊天**：您只需要在 Nexora Agent 的控制面板中批准该微信号的配对申请，对方此后才可以正常与 AI 对话。
- **全局开放**：如果您希望任何人随时都可以直接和 AI 对话，可以在配置中将权限限制调为“无限制”或者关闭白名单。

## 6. 绑定过程中的高频常见问题

### Q1：点击了“扫码绑定”，等了很久都没有二维码出来？
* **解决方法**：
  1. 请检查左上角的状态灯是不是 **🟢 绿色**。如果不是，必须先点启动。
  2. 检查您的电脑网络。公司局域网或校园网等特定网络环境可能会屏蔽微信机器人登录接口，尝试更换网络（比如连手机热点）再试。
  3. 点击「停止 Nexora Agent」，等待完全停止后再次点击「启动」，然后重新点击扫码。

### Q2：手机微信上已经提示登录成功了，但软件里的卡片还是显示“未绑定”？
* **解决方法**：这通常是网络数据回传有些许延迟。请耐心等待 15 到 30 秒。如果还是没有反应，可以重新点击一次卡片上的“扫码绑定”刷新状态，或重启一次本地服务。

### Q3：AI 机器人会不会泄露我的账号隐私和密码？
* **安全说明**：
  - Nexora Agent 绝对不会保存或收集您的微信密码。扫码登录生成的只是一段加密的本地运行凭证。
  - 所有的运行凭证和聊天记录都只存放在您自己的电脑上（默认路径为 `%USERPROFILE%\.openclaw`）。
  - **请保护好您自己电脑上的配置文件夹，千万不要将其发送给陌生人。**
