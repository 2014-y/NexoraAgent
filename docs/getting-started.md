# 📖 新手快速入门：从零到实现第一次 AI 对话

> **💡 提示**：本指南专为**完全没有任何编程基础、不会写代码**的普通用户编写。  
> 跟着以下步骤一步步操作，您只需花 **5 到 10 分钟** 就能让 AI 助手帮您自动回复消息。

无论您是想用它来帮自己分担日常繁琐的客服工作，还是想让它化身群聊气氛组，甚至是作为一个随叫随到的私人助理，Nexora Agent 都能帮您轻松实现。

---

## 🧭 在开始之前，我们需要了解的三个“关键角色”

在正式操作前，请先花一分钟了解这三个概念，这能帮您在后续配置时避免很多疑惑：

| 角色名称 | 它是做什么的？ | 您需要为它做什么？ |
| :--- | :--- | :--- |
| **Nexora Agent (桌面应用)** | 运行在您电脑上的“控制中心”或“调度中心”。它负责接收消息并指挥其他部分运转。 | 下载并安装它，然后点击界面上的「启动」按钮保持运行。 |
| **API Key (模型密钥)** | 这就像是 AI “大脑”的通行证。有了它，AI 才会思考和回答问题。 | 您需要去 AI 大模型官网注册并获取一串密钥，然后填入软件中。 |
| **聊天平台 (微信/飞书/QQ)** | AI 助手与您和朋友们见面的地方（聊天的入口）。 | 掏出手机扫一扫软件弹出的二维码，授权绑定即可。 |

> [!IMPORTANT]
> **记住一句话：**
> **Nexora Agent 软件负责在本地电脑上架起一座桥梁，而 API Key 负责赋予桥梁智慧。两者缺一不可，只有网关启动且填了 Key，AI 助手才能工作。**

---

## 🗺️ 核心极简流程（仅需 4 步）

整个上手过程可以高度概括为以下四步，非常简单：

```
[第 1 步：下载安装软件] ──> [第 2 步：一键启动等到绿灯亮起]
                                     │
                                     ▼
[第 4 步：手机扫码绑定微信] <── [第 3 步：填写 AI 模型密钥(Key)]
```

---

## 👣 详细操作步骤

### 第 1 步：下载并安装软件

1. 用浏览器打开官方最新发布包下载地址：
   [GitHub Releases 下载页](https://github.com/2014-y/ClawAI/releases)
2. 在页面中找到最新发布的版本，下载以 `.exe` 结尾的文件（例如：`Nexora Agent Setup.exe`）。
3. 下载完成后，双击运行该安装程序。
4. **注意**：如果 Windows 系统弹窗提示 *“Windows 已保护你的电脑”* 或者 *“未知发布者”*：
   - 请不用担心，这是未签名个人软件的正常安全提示。
   - 点击提示窗口中的 **「更多信息」** 链接。
   - 随后会显现出 **「仍要运行」** 按钮，点击它即可开始安装。
5. 在安装向导中，您可以一路点击“下一步”，也可以根据需求将安装路径修改到其他盘符（如 D 盘）。
6. 安装完成后，您的桌面上会出现一个帅气的 **Nexora Agent** 机器人图标。

---

### 第 2 步：启动本地服务

1. 双击桌面上的 **Nexora Agent** 图标打开软件。
2. 看到漂亮的管理界面后，找到左上角那个显眼的 **「启动 Nexora Agent」** 按钮，用鼠标点击它。
3. 此时，请观察按钮旁边的状态指示灯，它会经历如下变化：
   - 🔴 **红色**：服务目前处于停止状态，无法工作。
   - 🟡 **黄色**：系统正在检测本地依赖、载入后台插件。**此时请耐心等待，切记不要反复连续点击。**
   - 🟢 **绿色**：已就绪！说明您电脑本地的网关已经启动成功，可以开始接收连接了。

> [!NOTE]
> 在启动的过程中，界面中间的控制台日志区会滚动刷新一些系统运行日志，这是软件在为您打通后台服务，属于正常现象。只要最终指示灯变为**绿色**即可。

---

### 第 3 步：配置大模型 API Key（非常关键）

没有大模型的密钥，AI 助手就没有“大脑”，无法理解并回答您的问题。

1. 点击左侧导航栏中的 **「模型配置」**。
2. 在该页面中，您会看到可以添加不同的模型供应商（如 DeepSeek、阿里云百炼、智谱 AI 等）。您可以挑选您自己有账号或者常用的一家。
3. 将您在供应商官网申请到的 **API Key**（通常是一长串包含字母和数字的密匙）完整复制并粘贴到对应的输入框中。
4. 点击保存配置。

#### 💡 API Key 申请入口指南（新手参考）
不同大模型平台的界面和入口可能会有微调，您可以直接点击下方链接去注册并获取密钥：
* **DeepSeek 开放平台**：[去申请 Key](https://platform.deepseek.com/)
* **阿里云百炼（通义千问）**：[去申请 Key](https://dashscope.console.aliyun.com/)
* **智谱 AI 开放平台**：[去申请 Key](https://open.bigmodel.cn/)

> [!TIP]
> 申请到 Key 后，复制时请确保不要多复制了前后的空格，也不要遗漏了任何字符，否则 AI 会因为鉴权失败而无法回复。

---

### 第 4 步：用软件内的「模型会话」快速测试

在把 AI 接到微信或飞书之前，我们强烈建议您先在软件内部测试一下大脑是否好用：

1. 点击左侧菜单栏的 **「模型会话」**（或带聊天气泡图标的菜单）。
2. 在聊天界面的顶部，选择您刚才在「模型配置」里设置好的那个默认模型。
3. 在下方的输入框中输入一句简单的话，比如：`你好，请用一句话证明你已经可以正常工作了。` 然后发送。
4. **如果 AI 很快给出了合理的回答**：恭喜您！您的大模型配置完全正确，可以放心地去绑定微信等通讯软件了。
5. **如果软件报错或者没有响应**：请回到「模型配置」检查 Key 是否填错，或者查看您的网络是否顺畅。如果还是解决不了，请查看 [安装与排错指南](./install-guide.md)。

---

### 第 5 步：扫码绑定您的微信或飞书

1. 确认软件左上角的指示灯为 **🟢 绿色** 运行状态。
2. 点击左侧菜单栏的 **「通讯管理」**。
3. 找到您想要接入的平台（例如 **微信** 或者是 **飞书**）。
4. 点击卡片上的 **「扫码绑定」** 按钮。
5. 此时软件中间的日志区或弹出的窗口中会生成一个二维码。
6. 打开手机上的微信/飞书，使用自带的**扫一扫**功能扫描该二维码，并在手机端确认授权登录。
7. 回到电脑软件界面，您会发现卡片的状态已经神奇地变成了 **「已绑定」**，代表连接已经建立！

> [!IMPORTANT]
> 此时，您可以拿另一部手机或者让朋友给您刚扫码绑定的账号发送一句 `你好`，AI 就会接管并自动给出机智的回复啦！

## 🛠️ 日常维护与使用常识

- **别关软件**：因为本程序完全跑在您自己的本地电脑上以确保隐私，所以只要您希望 AI 助手替您自动回复，您的电脑就不能关机，且软件必须保持在 **绿灯** 运行状态。
- **让软件缩至后台**：如果您觉得窗口碍眼，可以点击窗口右上角的最小化，软件会自动缩入系统右下角的托盘图标中，在后台默默工作，不占用您的桌面空间。
- **开机自启**：您可以通过软件设置开启随电脑开机自启，免去每次手动打开的麻烦。
