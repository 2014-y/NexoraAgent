# 新手入门：从零到第一次对话

> 本文只面向**完全不会写代码**的用户。  
> 跟着做，大约 **5～10 分钟** 就能聊起来。

相关文档：

- [安装说明与排错](./install-guide.md)（装不上、报错时再打开）
- [微信 / 多渠道接入](./wechat-guide.md)（绑定微信用这篇）

---

## 开始前先弄清三件事

| 名称 | 是什么 | 你要操心吗 |
| :--- | :--- | :--- |
| **Nexora Agent** | 装在电脑上的桌面软件（本程序） | 要装、要点「启动」 |
| **大模型 API Key** | 调 AI「大脑」的通行证（如 DeepSeek） | **必须自己申请并填写** |
| **微信 / QQ 等** | 用来跟助手聊天的入口 | 扫码绑定即可 |

记住一句话：

**软件负责把消息接到电脑；API Key 负责让 AI 能回答。两者缺一不可。**

---

## 流程总览（只需 4 步）

```text
下载安装包 → 安装打开软件 → 点「启动」等到绿灯
      → 填 API Key →（可选）扫码绑定微信 → 开始聊天
```

---

## 第 1 步：下载并安装

1. 用浏览器打开下载页：  
   **https://github.com/2014-y/Nexora-Agent/releases**
2. 找最新版本，下载名字类似这样的文件：  
   **`Nexora Agent Setup 2.0.1.exe`**
3. 双击这个 `.exe` 文件。
4. 如果弹出 Windows 提示「Windows 已保护你的电脑」：
   - 点 **「更多信息」**
   - 再点 **「仍要运行」**
5. 安装向导里可以一路「下一步」，也可以把安装位置改到 D 盘。
6. 装好后，桌面会出现 **Nexora Agent** 图标。

> 若下载或安装失败，请看 [安装说明](./install-guide.md)。

---

## 第 2 步：打开软件并启动服务

1. 双击桌面的 **Nexora Agent**。
2. 看到主界面后，点左上角的 **「启动 Nexora Agent」**。
3. 等状态变成绿色（就绪）。

### 状态灯怎么看？

| 状态 | 意思 | 你可以做什么 |
| :--- | :--- | :--- |
| 🔴 红色 | 服务没开 | 点击「启动」 |
| 🟡 黄色 | 正在启动、加载插件 | **耐心等**，不要反复狂点 |
| 🟢 绿色 | 已经准备好 | 可以去填 Key、绑微信、聊天 |

启动过程中，中间的日志窗口可能会刷很多英文/中文行——这是正常现象，不必惊慌。  
等到绿灯，或日志里出现类似「就绪 / ready / 插件装载完毕」的字样，就可以继续。

---

## 第 3 步：填写大模型 API Key（必做）

没有 Key，助手无法思考和回复。

1. 点左侧菜单的 **「模型配置」**。
2. 按页面提示添加一个模型提供商（任选你常用的，例如 DeepSeek、通义千问、智谱等）。
3. 把你申请到的 **API Key** 填进去。
4. 保存配置。

### API Key 去哪里申请？（示例）

不同厂商界面会变，下面给常用入口（以官网最新说明为准）：

| 厂商 | 大致入口 |
| :--- | :--- |
| DeepSeek | https://platform.deepseek.com/ |
| 阿里云百炼（通义） | https://dashscope.console.aliyun.com/ |
| 智谱 AI | https://open.bigmodel.cn/ |

申请后一般会得到一串很长的字母数字，把它**完整复制**进 Nexora Agent，不要多空格、也不要截断。

---

## 第 4 步：用软件里的「模型会话」先试一下（推荐）

在绑定微信之前，建议先在软件内部试聊，确认模型可用：

1. 点左侧 **「模型会话」**（或类似「聊天」菜单）。
2. 顶部选好刚配置的模型。
3. 输入：`你好，请用一句话介绍你自己`
4. 如果 AI 正常回复 → 说明配置成功，可以继续绑微信。  
5. 如果报错 / 无响应 → 回到「模型配置」检查 Key、网络，或看 [安装说明 · 排错](./install-guide.md)。

---

## 第 5 步：绑定微信（可选，但很常用）

完整说明见：[微信接入教程](./wechat-guide.md)

超简版：

1. 确认状态灯是 **绿色**。
2. 打开 **「通讯管理」**。
3. 在 **微信** 卡片点击 **扫码绑定**。
4. 手机微信扫描屏幕二维码并确认。
5. 卡片显示 **已绑定** 后：
   - 用手机微信给这个号发消息（或用测试微信号），看是否自动回复。

> **重要**：电脑要保持 Nexora Agent **开着且绿灯**，微信通道才在线。关掉软件等于助手下班。

## 日常怎么用？

| 场景 | 操作 |
| :--- | :--- |
| 每天开机要用 | 打开 Nexora Agent → 点「启动」→ 等绿灯 |
| 暂时不用 | 点「停止」，或直接关窗口（可缩到托盘） |
| 换模型 | 「模型配置」里改默认模型 / Key |
| 看花了多少 | 打开「用量监控」 |
| 开联网搜索等能力 | 打开「内置插件」，找到对应卡片打开开关，然后**停止并重新启动**一次 |

## 记忆在哪里？（可选进阶）

助手可以把重要信息记在本地文件里，方便长期记住你的偏好。

常见位置（把 `你的用户名` 换成自己电脑登录名）：

```text
C:\Users\你的用户名\.openclaw\workspace\MEMORY.md
```

可用记事本打开，例如写上一行：

```text
- Yuan 喜欢美式咖啡，回复时语气简洁。
```

保存后，之后的对话更容易用到这些信息。

---

## 卡在这里？先对照这张表

| 现象 | 优先检查 |
| :--- | :--- |
| 软件打不开 / 闪退 | 杀毒软件拦截？重新安装？见 [安装说明](./install-guide.md) |
| 一直红灯 | 点启动；结束残留 `node.exe`；检查端口 18789 |
| 绿灯了但不回消息 | API Key 是否有效；网络是否通；微信是否真的「已绑定」 |
| 扫不出二维码 | 必须先绿灯；网络是否需要加速；稍后重试扫码 |
| 感觉很卡 | 换更快的模型，或关掉不需要的插件后再重启 |
